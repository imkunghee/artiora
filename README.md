# ARTILANA Lens

IP 데이터 기반 창작물 권리위험·시각품질 진단 서비스.

ARTILANA Lens는 생성형 AI 시대에 이미지·캐릭터·디자인 시안의 권리 위험, 공개 노출 위험, 시각 품질 문제를 사전에 점검하는 창작자/기업용 검수 도구입니다.

현재 목표는 “완전한 도용 탐지”가 아니라, 공개 IP 데이터와 시각 분석을 기반으로 위험 신호를 진단하고 제출 가능한 리포트를 만드는 MVP입니다.
